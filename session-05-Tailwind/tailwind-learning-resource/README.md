## ✨ How to use this resource
Each module contains **interactive-style sample codes**. You can copy-paste these into [Tailwind Play](https://play.tailwindcss.com/) to see them in action instantly!

### 💡 Pro Tip: Dark Mode Testing
For projects with Dark Mode, look for the toggle script in the solution code. You can click the moon/sun icon in the preview to switch themes.
